Hallo there!

My name is Charlotte Holmen, I'm a web developer from Stockholm that currently lives in Berlin. I started out this blog to try out some new features in web development, and at the same time I'll share some tips about Berlin.

So what features does this webpage contain so far?

Well, first of all I made a mobile first responsive design, with a left swipe sensitive menu. So it fits most modern devices and browsers. I'm testing on iOS devices, so I haven't checked if there's any funny looking Android features. I do the design as I go and just try to keep it clean, modular and scalable. One thing I thought was fun working on is the Google Map that changes depending on what location I'm writing about.

For developers

If you are a developer it could be interesting to know that I'm using Grunt, Require JS, SASS and Django CMS based on python. The next step now is to figure out a way to include Angular in this situation.

So that's what I'm working on right now. Enjoy reading!
